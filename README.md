# Backend Social API

Clean Architecture backend project built with:

- Node.js
- Express
- TypeScript
- dotenv

Course: PROG3271

---

## Setup Instructions

1. Clone the repository
2. Run:

npm install

3. Create a `.env` file in the root directory
4. Add:

PORT=3000

5. Start the server:

npm run dev

Server will run on:
http://localhost:3000
